📋 Descripción del Sistema

Sistema empresarial de gestión de compras de alta performance que incluye manejo de inventario multi-ubicación, gestión de pagos en cuotas con reglas de negocio avanzadas, sistema de promociones flexible, notificaciones automáticas vía email y cache Redis para optimización de rendimiento. Desarrollado con Django REST Framework siguiendo las mejores prácticas de arquitectura de APIs, incluyendo testing exhaustivo con Pytest, procesamiento asíncrono con Celery y monitoreo de performance en tiempo real.

✨ Características Principales

🐳 Arquitectura Containerizada

Sistema completo con Docker Compose orquestando múltiples servicios en producción.

  • Backend Django con Gunicorn
  • MySQL 9.0 con persistencia
  • Redis 7 para cache y broker
  • Celery Worker para tareas async
  • Celery Beat para cron jobs
  • Health checks automáticos

⚡ Sistema de Cache Redis

Cache inteligente con Redis 7-Alpine para máxima performance.

  • Cache de queries frecuentes
  • Invalidación automática por patrones
  • Fallback a memoria local
  • API de administración de cache
  • Monitoreo en tiempo real
  • Mejoras de hasta 95% en latencia

🧪 Testing Exhaustivo

Suite completa de tests automatizados con Pytest y cobertura >55%.

  • Tests unitarios y de integración
  • Factory Boy para fixtures
  • Ejecución paralela (pytest-xdist)
  • Coverage reports detallados
  • CI/CD integration ready
  • Mocking de servicios externos

🌿 Procesamiento Asíncrono

Celery + Redis para tareas en background sin bloquear requests.

  • Envío de emails asíncrono
  • Generación de reportes complejos

  • Reintentos automáticos
  • Monitoreo de tareas
  • Rate limiting inteligente
